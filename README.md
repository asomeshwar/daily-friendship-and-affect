# daily-friendship-and-affect
R code for the paper examining associations between daily close friendship experiences and daily affective states in adolescent girls.
## Contents
This repository contains supplementary analysis code accompanying the manuscript. All code is written in R.
## Data Availability
Questionnaire and daily diary data are available through the Research Domain Criteria (RDoC) database (reference number 2315): https://ndar.nih.gov/edit_collection.html?id=2315
Additional data are available from the PI on reasonable request.
## Citation
Please cite the associated manuscript when using this code.
